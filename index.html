FlexCap: Generating Rich, Localized, and Flexible Captions in Images

Debidatta Dwibedi, Vidhi Jain, Jonathan Tompson, Andrew Zisserman, Yusuf Aytar
Paper Video

Abstract

FlexCap is a module that generates flexible image descriptions. It can describe any selected region of an image and adjust the description length as desired. We train FlexCap to produce length-conditioned text describing a bounding box in the image. By using the length of textual description as a proxy for information density we enable controllable generation of the full spectrum of localized image descriptions, ranging from short object names tofull sentence captions. We explore several applications of this approach. First, FlexCap performs better at dense captioning as compared to baselines on the Visual Genome dataset. Second, by using FlexCap for visual grounding with an LLM for reasoning as FlexCap-LLM, we exceed the state-of-the-art zero-shot performance on visual question answering tasks. Finally, we qualitatively show the diverse applicability of FlexCap to image labeling, object attribute recognition,and visual dialog.
